### 常用指令 

面板(pane)

|   前缀   |  指令  |                            描述                            |
| :------: | :----: | :--------------------------------------------------------: |
| `Ctrl+b` |   ?    |                          显示帮助                          |
| `Ctrl+b` |   z    |                      面板最大化/恢复                       |
| `Ctrl+b` |   %    |                        水平新建面板                        |
| `Ctrl+b` |   "    |                        垂直新建面板                        |
| `Ctrl+b` | 方向键 |                          切换面板                          |
| `Ctrl+b` |  `x`   |         关闭当前面板（关闭前需输入`y` or `n`确认）         |
| `Ctrl+b` |  `;`   |                  切换到最后一次使用的面板                  |
| `Ctrl+b` |  `q`   | 显示面板编号，在编号消失前输入对应的数字可切换到相应的面板 |
|          |        |                                                            |
